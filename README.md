CE-ESY Assignments
مستودع لرفع واجبات مادة النظم المضمنة - سنة رابعة هندسة حواسيب.

الواجب الأول: تنفيذ الـ Circular Buffer
في هذا المجلد، قمت بكتابة كود بلغة C يطبق فكرة المخزن الدائري (Circular Buffer) من الصفر بدون الاعتماد على أي مكتبات جاهزة لضمان فهم طريقة عمل المؤشرات يدوياً.  

تفاصيل الحل:
الملف الأساسي هو prog.c، وفيه قمت بتعريف الـ struct الخاص بالمخزن مع المؤشرات الأساسية (head_ptr و tail_ptr).  

الكود يعالج المشاكل البرمجية المعتادة في هذا النوع من المخازن مثل حالة الامتلاء (Overflow) وحالة الفراغ (Underflow) مع إظهار رسائل تنبيه واضحة.  

للتأكد من صحة العمل، جربت إدخال اسمي مع اللاحقة المطلوبة CE-ESY واستخدمت أحجام مختلفة للـ Buffer لاختبار استقرار البرنامج في كل الحالات.
